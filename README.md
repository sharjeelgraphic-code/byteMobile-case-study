# byteMobile-case-study
React Native case study showcasing my UI development work for a Saudi Arabia-based mobile application.


 Project Overview
byteMobile is a mobile application developed for users in Saudi Arabia. The app was built using React Native with a focus on delivering a modern, responsive, and user-friendly mobile experience.

 My Role
I worked as a React Native Developer and was responsible for:

- Converting Figma designs into mobile screens
- Building reusable UI components
- Implementing navigation flows
- Creating responsive layouts
- Optimizing UI performance
- Fixing bugs and improving user experience

 Technologies Used

- React Native
- JavaScript / TypeScript
- Expo
- React Navigation
- Redux / Context API
- REST APIs
- Figma
